# Aviasales System Dashboard

## Requirements

- python >= 3.6
- pipenv >= 2018.11.26
- node >= 8.10
- npm >= 5.6

## Installation

1. git clone git@github.com:hutsi/aviasales-test.git
2. cd aviasales-test
3. cd backend
4. pipenv install
5. pipenv shell
6. FLASK_APP=server.py flask run (make sure server is running on http://127.0.0.1:5000)
7. cd ./../frontend
8. npm i
9. npm start
10. open http://127.0.0.1:8030 in browser
11. enjoy :)
