## Requirements

- python >= 3.6
- pipenv >= 2018.11.26
- node >= 8.10
- npm >= 5.6

## Installation

1. `git clone git@github.com:hutsi/aviasales-test.git`
2. `cd aviasales-test/backend`
3. `pipenv install`
4. `pipenv shell`
5. `FLASK_APP=server.py flask run` (make sure server is running on http://127.0.0.1:5000)
6. `cd ./../frontend`
7. `npm i`
8. `npm start`
9. open `http://127.0.0.1:8030` in browser
10. enjoy :)
